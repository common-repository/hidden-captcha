Bloquea el spam de todo tipo de robots con un captcha que es invisible para los usuarios de tu web o blog.

== Description ==

El hidden captcha es un plugin anti-spam que permite bloquear a los robots que pretenden incluir spam en nuestra web o blog de forma abusiva y descontrolada.
Con este plugin los usuarios no tendr&aacute;n que rellenar ning&uacute;n tipo de captcha, ya que para ellos el captcha se hace invisible, siendo visible exclusivamente para los robots.

Ventajas

*   Evita spam en comentarios.
*   Protege el acceso de login.
*   Protege el acceso de registro de usuarios.
*   Protecci&oacute;n en el formulario de p&eacute;rdida de password.
*   No utiliza javascript por lo que la velocidad de carga no se ver&aacute; afectada.
*   Totalmente oculto para el usuario.

M&aacute;s informaci&oacute;n en www.elplugin.com/hidden-captcha/

== Installation ==

Para instalar el plugin:

*   Descargar el plugin, descomprimir el archivo y subir la carpeta al directorio /wp-content/plugins/
*   Activar el plugin.
